## Инструкция по запуску проекта

Скачать или склонировать репозиторий

```bash

# установка зависимостей
$ npm install

# запуск MongoDB
$ mongod

# запуск сервера с "горячей" перезагрузкой на localhost:3000
$ npm run dev

# запуск сервера на localhost:3000
$ npm run start

```
